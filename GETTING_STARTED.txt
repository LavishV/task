===============================================
QUICK START (5 MINUTES)
===============================================

1. SETUP DATABASE

   Open PostgreSQL:
   psql -U postgres

   Copy-paste this:
   CREATE DATABASE realestate_db;
   CREATE USER realestate WITH PASSWORD 'password123';
   ALTER ROLE realestate WITH CREATEDB;
   GRANT ALL PRIVILEGES ON DATABASE realestate_db TO realestate;
   \q

2. INSTALL DEPENDENCIES

   npm install

3. START BACKEND (Terminal 1)

   npm run server

   Wait for: "Server is running on http://localhost:5000"

4. START FRONTEND (Terminal 2)

   npm run dev

   Wait for: "Local: http://localhost:5173/"

5. OPEN IN BROWSER

   Landing Page: http://localhost:5173/
   Admin Panel: http://localhost:5173/admin

===============================================
TROUBLESHOOTING
===============================================

Can't connect to database?
  - Check PostgreSQL is running
  - Verify .env credentials
  - Run: psql -U postgres -l

Port already in use?
  - Change PORT in .env to 5001
  - Or kill the process using lsof or netstat

Images not uploading?
  - Create uploads folder: mkdir uploads
  - Check folder permissions: chmod 755 uploads

Still having issues?
  - Read SETUP.md completely
  - Check browser console (F12)
  - Check terminal for error messages
